Conditional Logic: `if`, `elif`, `else`

Conditional statements are the backbone of decision-making in programming. They allow your code to execute different paths based on whether certain conditions are met. Python uses `if`, `elif` (short for "else if"), and `else` to define these paths. The conditions are typically evaluated using comparison operators like `==`, `!=`, `<`, `>`, `<=`, and `>=`. You can also combine multiple conditions using logical operators like `and` and `or`. 

x = 10
if x > 15:
    print("x is greater than 15")
elif x > 5:
    print("x is greater than 5 but not 15")
else:
    print("x is 5 or less")

Loops and Iteration: `for` and `while`

Loops are fundamental for performing repetitive tasks. The `for` loop is used to iterate over a sequence (like a list, string, or tuple) or any other iterable object. The `range()` function is often used with `for` loops to generate a sequence of numbers, providing a simple way to loop a specific number of times. The `while` loop, on the other hand, continues to execute a block of code as long as a specified condition remains `True`.

`range(start, stop, step)`: This function generates a sequence of numbers. `start` is inclusive, `stop` is exclusive, and `step` is the increment. 

# for loop with range
for i in range(5):  # Generates 0, 1, 2, 3, 4
    print(i)

# while loop
count = 0
while count < 3:
    print("Counting...")
    count += 1

The `datetime` Module

The `datetime` module is Python's standard library for working with dates and times. It provides classes for manipulating dates and times, handling time zones, and performing arithmetic operations like adding or subtracting days. It is a crucial tool for any application that needs to manage time-related data. You can find detailed documentation for the module here. 

import datetime

# Get today's date
today = datetime.date.today()
print(f"Today's date: {today}")

# Add a time delta of 7 days
future_date = today + datetime.timedelta(days=7)
print(f"Date after 7 days: {future_date}")
